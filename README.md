# Fundamentos y Lógica Java
En este repositorio se documenta mi aprendizaje aplicado en conceptos fundamentales y logica de Java. Desde los conceptos más básicos hasta la programación orientada a objetos, aplicando la teoría en retos de código reales.

## Objetivos
* Practicar y dominar la sintaxis de Java.
* Fortalecer la lógica de programación.
* Documentar mi proceso para construir una base fuerte para frameworks como Spring Boot.

## Estructura del Repositorio
Cada carpeta representa un módulo que contiene ejercicios prácticos:

* 1. Learn the Basics (Fundamentos Básicos)
* 2. Object Oriented Programming (POO)
* 3. Advanced Concepts & Features
* 4. Herramientas de Java moderno.


## Herramientas utilizadas
* **Lenguaje:** Java 17
* **IDE:** IntelliJ IDEA
